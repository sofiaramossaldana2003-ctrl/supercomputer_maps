# 🚀 Evolución de la Supercomputación Global (1993–2021)

Análisis exhaustivo del ranking TOP500 para identificar tendencias geopolíticas y soberanía tecnológica a través de la infraestructura de High-Performance Computing (HPC).

## 📊 Sobre el Proyecto
El HPC no es solo potencia de cálculo; es la columna vertebral de la **Inteligencia Artificial**, las simulaciones científicas y la investigación estratégica nacional. Este proyecto procesa más de **28,500 registros históricos** para mapear cómo ha cambiado el dominio tecnológico mundial desde 1993.

## 🛠 Metodología
A diferencia de los análisis estándar que se limitan a snapshots anuales, este proyecto utiliza el **acumulado histórico** para:
* Suavizar la volatilidad de las inversiones anuales.
* Identificar la inercia tecnológica y la capacidad instalada a largo plazo.
* Contrastar el crecimiento de potencias como EE. UU. 🇺🇸, Japón 🇯🇵 y China 🇨🇳 ante la evolución de los fabricantes clave.
